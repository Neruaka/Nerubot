# Nerubot
Description
Nerubot est un bot Discord développé en Node.js, conçu pour faciliter la gestion et l'animation des serveurs Discord. Ce projet est destiné à aider les administrateurs de serveurs à automatiser des tâches récurrentes et à offrir des fonctionnalités interactives aux membres.

Fonctionnalités
Listez ici les principales fonctionnalités de Nerubot, par exemple :

Gestion des rôles : automatise l'attribution des rôles aux utilisateurs selon des critères prédéfinis.
Modération : offre des outils pour la modération des discussions, comme la suppression de messages ou le bannissement d'utilisateurs indésirables.
Interaction : réagit aux commandes des utilisateurs par des réponses ou des actions programmées.
Prérequis
Pour utiliser ce bot, vous aurez besoin de :

Node.js (version X.X ou supérieure)
npm (Node Package Manager)

Configuration
Pour configurer votre bot, vous devez créer un fichier .env à la racine du projet et y définir les variables suivantes :

DISCORD_TOKEN: Le token de votre bot Discord.
OTHER_VARIABLE: Autre variable nécessaire.
Exemple de contenu pour le fichier .env :

makefile
Copier le code
DISCORD_TOKEN=votre_token_ici
OTHER_VARIABLE=valeur
